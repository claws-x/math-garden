# Erdős–Straus 猜想（Erdős–Straus Conjecture）

## 基本信息

| 属性 | 内容 |
|------|------|
| 名称 | Erdős–Straus 猜想 |
| 提出者 | Paul Erdős & Ernst G. Straus（1948） |
| 领域 | 数论 / 丢番图方程 |
| 状态 | ⏳ 未解决 |
| 分级 | B+ 级 / 二级重点 |
| 档案建立 | 2026-03-25 |

---

## 1. 一句话描述

对每个大于 1 的整数 n，4/n 都可以写成三个单位分数之和吗？

---

## 2. 严谨表述

对所有整数 n ≥ 2，方程

4/n = 1/x + 1/y + 1/z

是否总有正整数解 (x, y, z)？

---

## 3. 为什么重要

1. **描述极其简单**：任何人都能理解——把 4/n 拆成三个"一份之几"的和
2. **与古埃及分数的联系**：古埃及人只使用单位分数（1/k），这个猜想是现代版的"埃及分数问题"
3. **数论中的经典问题**：涉及丢番图方程的可解性
4. **计算验证很远但证明很难**：已验证到 n = 10¹⁴ 以上，但理论证明遥遥无期
5. **Erdős 标志性问题**：Erdős 本人最关注的问题之一

---

## 4. 已知进展

### 计算验证
- 已对所有 n ≤ 10¹⁴ 验证成立
- 没有找到反例

### 理论结果
- **密度结果**：满足猜想的 n 的密度为 1（即"几乎所有"n 都满足）
- **模条件**：对 n 满足特定同余条件时可以证明（覆盖大部分 n）
  - 例如 n ≡ 0 (mod 4)：4/n = 1/n + 1/n + 2/n → 三项分解存在
  - 类似的模条件可以覆盖大部分情况
- **例外集**：只需证明 n 为某些特殊素数时成立
- **Schinzel 结果**：将问题约化到 n 为素数的情况

### 推广
- Sierpiński 猜想：5/n = 1/x + 1/y + 1/z 对 n ≥ 2 成立
- 一般问题：a/n 何时可以写成 k 个单位分数之和？

---

## 5. 难点分析

1. **素数情形最难**：偶数和大部分合数的情况都已解决，剩下的都是"顽固素数"
2. **缺乏统一方法**：现有证明都是分模条件处理，无法统一
3. **丢番图方程的本质困难**：方程 4n = xy + xz + yz 的解的存在性没有通用判据
4. **"最后 1%"问题**：虽然"几乎所有"n 都满足，但证明"所有"需要质的飞跃

---

## 6. 常见误解

### ❌ "计算验证就等于证明"
即使验证到 10¹⁴ 也不能排除更大的反例。数论中有很多猜想在极大数值处才出现反例。

### ❌ "这个问题很简单因为描述很简单"
描述简单不等于证明简单。这是数论的典型特征。

---

## 7. 普通人理解方式

想象你有 4 块蛋糕要平均分给 n 个人。你不能直接切成 n 份——你只能切成"1/某个整数"的大小。

问题是：你能不能只切**三刀**（三种大小的块），就能凑出 4/n？

猜想说：不管 n 是多少，三刀永远够用。
